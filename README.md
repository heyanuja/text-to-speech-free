# text-to-speech-free

paste text and it reads it back and thats about it

speechify wanted money for, basically, a `<textarea>` and the web speech api, so.

## use

open `index.html` in chrome or safari, paste, pick a voice, hit play.

on mac the **(premium)** voices sound much better than the defaults — `system settings → accessibility → spoken content → manage voices` and download whichever sounds least robotic. they show up in the dropdown next time you open the page.

## keys

- `space` — play / pause
- `esc` — stop
- click any word to jump there
